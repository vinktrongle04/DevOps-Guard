# business-case/

**Đây là nơi chứa tài liệu thuyết phục — pitch, ROI, demo, product requirements.**

Load khi có câu hỏi về business value (Layer 2 — Load on demand).

| File | Mục đích | Đọc khi nào |
|---|---|---|
| `DEMO_SCRIPT.md` | Kịch bản demo live 3 phút | Trước/trong khi demo |
| `PITCH_DECK.md` | Outline 8 slide thuyết trình | Chuẩn bị presentation |
| `ROI_METRICS.md` | Phân tích ROI với nguồn số liệu | Giải thích cost savings |
| `ENTERPRISE_ADOPTION.md` | Setup guide + compliance story cho doanh nghiệp | Pitch cho CTO/Security Lead |
| `PRD.md` | Product Requirements Document | Giải thích yêu cầu sản phẩm |

## Thêm tài liệu business mới ở đây

Ví dụ:
- Case study khách hàng → thêm `CASE_STUDY.md`
- Pricing model → thêm `PRICING.md`
- Partnership proposal → thêm `PARTNERSHIP.md`
- Sau đó cập nhật routing table trong `docs/core-rules/AGENT_CONTEXT.md`
