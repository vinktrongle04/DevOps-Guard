# how-it-works/

**Đây là nơi giải thích kiến trúc và phương pháp đo lường.**

Load khi cần giải thích cho người ngoài (Layer 1 — Load on reference).

| File | Mục đích | Đọc khi nào |
|---|---|---|
| `TRAE_INTEGRATION.md` | Giải thích TRAE điều phối 5 gates như thế nào | Demo kỹ thuật, technical judge |
| `METHODOLOGY.md` | Tại sao hybrid architecture + nguồn gốc các con số | Giám khảo hỏi về số liệu |

## Thêm tài liệu kỹ thuật mới ở đây

Ví dụ:
- Giải thích SARIF output format → thêm `SARIF_GUIDE.md` vào đây
- Giải thích cách custom rule → thêm `CUSTOM_RULES.md` vào đây
- Sau đó cập nhật routing table trong `docs/core-rules/AGENT_CONTEXT.md`
